SQL有多少种join方式

Posted on 2018-12-12 Edited on 2019-11-26 In DataBase Views:

SQL JOIN 用于把来自两个或多个表的数据或行基于它们的公共字段结合起来。有四种不同的JOIN方式:

  • INNER JOIN
  • LEFT JOIN
  • RIGHT JOIN
  • FULL JOIN

在这里在线SQL网页里面有两张表，students表
